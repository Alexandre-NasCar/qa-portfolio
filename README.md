## Tipos de testes

### Testes manuais

- testes funcionais;
- testes exploratórios;
- testes de interface;
- testes de responsividade;
- testes de regressão;
- validação de regras de negócio;
- análise de requisições e respostas no DevTools.

### Automação Web

- Cypress;
- Playwright;
- Selenium WebDriver;
- validação de fluxos de usuário;
- testes End-to-End;
- execução em diferentes resoluções.

### Testes de API

- Postman;
- REST Assured;
- validação de status HTTP;
- análise de payloads e respostas;
- testes positivos e negativos;
- validação de autenticação e regras de negócio.

### Testes de performance

- k6;
- Apache JMeter;
- análise de tempo de resposta;
- taxa de erros;
- throughput;
- usuários virtuais;
- execução de cenários de carga.

### BDD

- Cucumber;
- Gherkin;
- escrita de features;
- cenários com `Given`, `When` e `Then`;
- documentação orientada ao comportamento.
