# Portfólio de Quality Assurance

Portfólio profissional dedicado à documentação de projetos, testes de software, automações, evidências, relatórios e aprendizados em Quality Assurance.

## Objetivo

Demonstrar, de forma organizada e rastreável, minha atuação prática em testes manuais, automação Web, testes de API, testes de performance, BDD, análise de resultados e documentação da qualidade.

## Mapa do portfólio

| Área | Conteúdo |
|---|---|
| Projetos | Aplicações e contextos utilizados nos testes |
| Testes manuais | Casos de teste, execuções e evidências |
| Automação Web | Cypress, Playwright e Selenium |
| Testes de API | Postman e RestAssured |
| Performance | k6 e JMeter |
| BDD | Features e cenários Gherkin |
| Relatórios | Bugs, execuções e resumos executivos |
| LinkedIn | Publicações conectadas aos projetos |

## Projetos em destaque

### PackAndGo

Projeto educacional e autoral utilizado para aplicação de práticas de desenvolvimento e Quality Assurance.

### E2E Treinamentos

Experiências práticas em testes de software, apresentadas com respeito à privacidade, confidencialidade e proteção de informações internas.
